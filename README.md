> Uma página web moderna para a biblioteca virtual da UniFECAF, com design responsivo e interativo.

## 🎯 Visão Geral

Este projeto consiste em um site institucional para a **Biblioteca UniFECAF**, desenvolvido com HTML, CSS e JavaScript puro. O layout segue um padrão moderno com seções bem definidas, cores alternadas e cards interativos para exibição de livros.

## ✨ Funcionalidades

- ✅ **Header fixo** com menu de navegação suave
- ✅ **Banner principal** com chamada visual
- ✅ **Seção "Sobre"** com informações institucionais
- ✅ **Destaque visual** com imagem central de grande impacto
- ✅ **Grade de livros** (3x2) com cards contendo:
  - Imagem ilustrativa
  - Título da obra
  - Sistema de avaliação por estrelas (⭐)
  - Descrição resumida
  - Botão interativo "Ver mais"
- ✅ **Seção de novidades** com fundo escuro e itens lado a lado
- ✅ **Área de contato/informações** institucionais
- ✅ **Rodapé** com direitos autorais
- ✅ **Design responsivo** adaptado para tablets e smartphones
- ✅ **Scroll suave** entre as seções
- ✅ **Favicon** personalizado com emoji de livro

## 🛠️ Tecnologias Utilizadas

| Tecnologia | Finalidade |
|------------|------------|
| **HTML5** | Estrutura semântica do site |
| **CSS3** | Estilização, layout flexbox/grid e responsividade |
| **JavaScript (Vanilla)** | Renderização dinâmica de cards, interatividade e scroll suave |
